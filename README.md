# 🧹 Home Cleaning Assistant

<p align="center">
  <b>A smart web-based system for managing home cleaning services, bookings, and employee assignments.</b>
</p>

---

## 📌 Description
The **Home Cleaning Assistant** is a web-based application that simplifies the process of booking cleaning services, assigning employees, and tracking tasks efficiently.  
It is built using **Java (IntelliJ IDEA)** for backend and a responsive frontend using **HTML, CSS, and Bootstrap**.

---

## 🚀 Features
✨ User-friendly service booking  
👨‍💼 Admin employee assignment system  
📊 Task tracking & status updates  
📱 Fully responsive UI  

---

## 🛠️ Tech Stack
- ☕ Java (Backend)
- 🌐 HTML, CSS, Bootstrap (Frontend)

---

## ▶️ How to Run
1. Open project in **IntelliJ IDEA**  
2. Run the Java backend server  
3. Open browser and access the application
