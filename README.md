# 🧹 Home Cleaning Assistant

## 📌 Description
A web-based Home Cleaning Assistant system that simplifies service booking, employee assignment, and task management. Built using Java (IntelliJ IDEA) with a responsive frontend using HTML, CSS, and Bootstrap.

---

## 🚀 Features
- User can book cleaning services  
- Admin can assign employees  
- Task tracking and status updates  
- Responsive UI using Bootstrap  

---

## 🛠️ Tech Stack
- Java (Backend)
- HTML, CSS, Bootstrap (Frontend)

---

## ▶️ How to Run
1. Open project in IntelliJ IDEA  
2. Run the Java backend server  
3. Open browser and access the application
